Manage links to your tags manually from the editor.

== Description ==

MCE Tag Button allows you to retrieve all your tags in a modal box and generate links to this tags in your content. Select a part of your text you want to link. Click on the new button in your editor. Select the tag you want to apply. Click ok to generate the link in your content.

*Feature List*
* Retrieve all your tags previously generated
* Put request of your tags in a cache file to speed up execution time
* Generate link to the tag page

== Installation ==

Step 1. Download the Plugin from .

Step 2. Upload it in you wp-content/plugins directory.

Step 3. Activate it from Admin panel.

Step 4. Go to your editor and start using it !.
